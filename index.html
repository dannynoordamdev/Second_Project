CSS styling starts by placing a selector, a property and value.

These are the most common selectors:

  • Universal selector: *
  • Type selectors: 'div', 'p', etc.
  • Class selectors: 'div class= "Sell-button"'. Styling uses a dot.
  • ID selectors: 'div id= "title"' Styling uses a #.
  • Groupped selectors: by writing .read, .unread { ... }
  • Chaining selector: '.subsection.header { ... }'

Basic CSS properties:

  • Color: #ffffff, rgb(100, 0, 127), hls(15, 82%, 56%)
  • font-family
  • font-size
  • font-weight
  • text-align: center, right or left.
